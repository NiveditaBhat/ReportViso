ReportViso Features

Summary Report

Gives a brief overview of the report with the quality indicator. Summary report gives an idea to the end users about the report accuracy and the percentage of errors in the report

Detailed Report

Complete report with all the properties are displayed in tabular format

2D Viewer

Weld data represented as a scatter plot graph with interactive panel. Interactive setup helps end users to apply filter on weld properties like filter weld points based on STPOS errors, their geometrical type etc

Dashboard

Gives an overview of the user activity and quality of the saved user reports
