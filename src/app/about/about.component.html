Why ReportViso ?

ReportViso is a web based tool designed to view, visualize and analyze a certain type of weld report known as MR Report.To Learn more about MR Reports, Please click here

Traditionally, MR reports are viewed on CAD (Computer Aided Design) software like NX which are licensed and sometimes slow while displaying heavy content. ReportViso is developed with open source JavaScript Frameworks like angular, node, d3 which are fast and responsive. Itâ€™s not heavy on your computer hardware and is freely available. It is no replacement to CAD but a first step towards developing web based visualizing tools independent of CAD.

Features
  • Dashboard with user activity and report quality indicator
  • Upload MR Reports and view it as
    1. Summary Report - Gives an overview of the report with the quality indicator.Summary report gives an idea to the end users about the report accuracy and the percentage of errors in the report
    2. Detailed Report - Complete report with all the properties are displayed in tabular format
    3. 2D Viewer - Weld data is represented as a scatter plot graph with interactive panel. This helps the users to apply filter on weld properties like filter weld points based on STPOS errors, their geometrical type etc. This is the section where 2D visualization of the MR report can be done.
  • Save Reports and View them anytime
  • Delete Reports
